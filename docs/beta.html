    • Why have a Beta?
    It's simple! (kinda)
    Having the website be labeled as a "beta" is due to multiple reasons. One of them being that this website is under development! As things are being added to catch up to the full content, there are things missing and if it was a final version with things missing, it wouldn't be right.

    Being labeled a beta also allows people to freely edit / change and or update pages! As Honkai: Star Rail is still new and content is being added, the website will evolve to make sure all the content is added and accessable. (Just might take a little bit! (Solo dev moment but with some help!))

    (Your help will be much appreciated!)

    Please feel free to visit the GitHub page and add some new content!

    Version Breakdown


    Broken up into 5 sections:

    Release

    Main Version

    Sub Version

    Patch

    Hot-Fix

    Example:

    Beta

    v1

    .3

    -15

    -1


    Release

    In the example, "Beta" means that the website version is being worked on.

    Main Version

    In the example, "v1" means that the website is in it's first full version.

    Sub Version

    In the example, ".3" means that from the version, this is the 3rd main update to that version.

    Patch

    In the example, "-15" means that there have been 15 small updates to the Sub Version.

    Hot-Fix

    In the example, "-1" means that there has been 1 Hot-Fix on the patch.
